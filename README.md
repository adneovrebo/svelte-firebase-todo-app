# Svelte Firebase Todo

Testing out Svelte 3 with firebase

## Get started

Install the dependencies...

```bash
cd svelte-firebase-todo-app
npm install
```

...then start [Rollup](https://rollupjs.org) dev server:

```bash
npm run dev
```

## Building and running in production mode

To create an optimised version of the app:

```bash
npm run build
```
